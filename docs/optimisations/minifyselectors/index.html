minifySelectors

Removes unnecessary qualified universal selectors, unquotes attribute selectors, trims & normalizes selector strings.

Example

Input

*.box
.box::before
.box       .box
[class*=&quot;box&quot;]
.box ~ [class] {
    color: red;
}

Output

.box
.box:before
.box .box
[class*=box]
.box~[class] {
    color: red;
}
