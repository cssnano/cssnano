# postcss-normalize-casing

> Normalize casing of style declarations

```css
a { COLOR: 'RED'; }
```

```css
a { color: 'red'; }
```

## Usage

```js
postcss([ require('postcss-normalize-casing') ])
```

See [PostCSS] docs for examples for your environment.
